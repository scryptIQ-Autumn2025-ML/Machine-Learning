Machine Learning 1 - Glossary

Glossary ML1
Random Forest Classifier: An ensemble learning method that fits a number of decision tree classifiers on various sub-samples of the dataset and uses averaging to improve predictive accuracy and control over-fitting.
Decision Tree: A non-parametric supervised learning method used for classification and regression that predicts the value of a target variable by learning simple decision rules inferred from the data features.
Ensemble Learning: A machine learning technique that combines the predictions from multiple models to create a more accurate and robust prediction.
Averaging Methods: Build estimators independently and average their predictions (e.g., Random Forest).
Boosting Methods: Build estimators sequentially, where each new model attempts to correct the errors of the previous ones (e.g., Gradient Boosting).
State Space: The set of all possible configurations of a system. In classification visualisation, this often refers to the coordinate grid of all possible feature value combinations used to visualise decision boundaries.
Prediction Probability: The confidence score (between 0 and 1) assigned by the model to a prediction, indicating the likelihood that a sample belongs to a specific class.
Feature Importance: A score indicating how useful or valuable each feature was to the model in making predictions.
Over-fitting: A modelling error that occurs when a function is too closely fit to a limited set of data points. An over-fitted model captures noise rather than the underlying pattern, performing well on training data but poorly on test data.
Stratified Shuffle Split: A cross-validation object that returns stratified randomised folds. It creates splits of data into training and test sets while preserving the percentage of samples for each class (stratification).
ROC Curve (Receiver Operating Characteristic): A graphical plot that illustrates the diagnostic ability of a binary classifier system as its discrimination threshold is varied. It plots the true positive rate against the false positive rate.
AUC (Area Under the Curve): The computed area under the ROC curve. It represents the degree of separability or performance of the classifier; an AUC of 1.0 represents a perfect model, while 0.5 represents a model with no discrimination capacity (random guess).
