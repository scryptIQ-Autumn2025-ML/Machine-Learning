    Assignment ML2

    Assignment Background:

    This assignment makes use of the breast cancer dataset which can be imported from the 'scikit-learn'.

    The breast cancer data can be imported from the scikit-learn datasets library.

    Assignment Questions

    Note: Both the dataset and the classifiers used in this assignment have been filled out already.
    The linear Support Vector classifier is imported with the keyword argument dual=False. This is to reduce the number of warnings that occur when the classifier struggles to find a good solution. See the documentation for more information.

    Question 1

    Explore the dataset to get an understanding of the features:

    • Create a summary boxplot plot of all the features to see their medians and distributions.

    • To do so, first make the X data into a Pandas Dataframe, using a method to create the plot
    _(Hint! Look back to the Data Handling lessons for a reminder of how to do this)._

    Question 2

    Train the given classifiers and compare the results:

    • Split the dataset into train / test.

    • Use the test_train_split() function from sklearn.

    • Set the test dataset to 95% to make the classification task difficult.

    • Obtain the recall scores and print to screen formatted.

    Question 3

    Generate confusion matrices for each classifier:

    • The plot_confusion_matrix() function from the ML2 lesson materials is given for you.

    • Use this function to create a multi-plot containing a confusion matrix for each classifier.

    • Compare the confusion matrix with the recall score.

    • Can you see issues with the model that recall score fails to inform of?

    Question 4

    Perform a permutation test for each classifier:

    • Use the permutation_test_score() function.
      • Set the scoring parameter to "accuracy".

      • To increase the speed you can reduce n_permutations or you can set n_jobs to -1 to increase CPU core usage (see lesson).

    • Plot the permutation scores with additional lines indicating chance accuracy and the classifier score.

    Question 5

    Normalise the features and repeat Questions 1 - 4:

    • Use sklearn's Normalizer to augment the dataset.

    • Repeat the training, confusion matrices and permutation test.

    • Observe any differences / improvements.

    Question 6

    1. Perform hyperparameter tuning with the Random Forest classifier:
    A.

    Generate a SVC model and train it using the previously used training split already generated.
    • Print the score.
    (Note: this is different to LinearSVC which was used previously, as it allows all types of SVC kernels)

    B.

    Setup GridSearchCV to search for the best parameters for the classifier.
    • Use StratifiedShuffleSplit for the cross-validation argument (choose your own parameters).

    • Include the parameters kernel, C, gamma.

    • Give at least 3 different parameters for each one.

    • See the documentation for the default values / options.
    C.

    Run the grid search with both the original dataset and the normalised dataset:
    • Print the best parameters for both datasets.

    • Are there any differences between the datasets?
    Note! Given the low options in the grid search the result may be worse than the default settings. In fact the default settings can generally be quite good

    Question 7

    scikit-learn has many pre-processing functions, each with their specific use case, such as StandardScaler, MinMaxScaler and RobustScaler.

    • Read about their use cases in the linked documentation and try them out on the dataset.

    • Plot the augmented data.

    • Run and compare the parameter grid search for each dataset
    Head to the Lecture_Resources folder to scalers_demo.ipynb for examples of the three aforementioned pre-processing functions.
